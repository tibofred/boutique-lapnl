An easy, automated and advanced Zoho Wordpress web form generator to capture leads. Also sync Wordpress member registration as Contacts.

== Description ==
Zoho Wordpress integration is a plugin for capturing leads from web form page/post and contacts from user registration to CRM Zoho directly from WordPress. Zoho Wordpress integration plugin is now part of [WP Leads Builder For Any CRM](https://wordpress.org/plugins/wp-leads-builder-any-crm), a combined plugins for all CRM  plugins with more features. Zoho Wordpress plugin is a simple, advanced, easy to set up CRM Zoho web forms generator for Wordpress on the fly. It helps to generate a web form via user friendly plugin interface which in turn used as a contact form to convert your traffic/visitors as high potential leads to CRM Zoho CRM.

- No HTML
- No Manual Work
- Simple Set up
- User friendly UI
- More Options
- Short Code Featured
- Dynamically Generated
- Add, Edit fields at any time
- Easy Manageable

**Features**
*    Fetch CRM Zoho lead fields directly into Wordpress to create a form
*    Enable/disable fields to include in form or not
*    Short code to integrate form in post / page
*    Separate short code to integrate form even as widget site wide
*    Capture/sync new registered WP members to Zoho CRM contacts.

== Installation ==

1. First, install [CRM WordPress Leads](https://wordpress.org/plugins/wp-leads-builder-any-crm)
2. Then Extract the wp-zoho-crm.zip in wordpress/wp-content/plugins using FTP or through plugin install in wp-admin.
3. Activate the plugin in the WordPress plugin list.

== Frequently Asked Questions ==

1) How to create form with Contact Form 7?<br />

* Enable the Contact Form 7 in CRM configuration under Form Type.
* In CRM Forms menu select the required fields and Save.
* The shortcode with selected fields created in Contact Form 7.
* Place the shortcode in Post, Page or Widget.

2) Why I'm not able to submit the form created with Contact form?<br />

* After generating the shortcode for Contact form.
* Go to Contact â†’ Contact Forms â†’ Edit view of the shortcode.
* In Mail menu, configure all the settings and save.
* If the mail is not configured the form will trigger an error message stating 'Submission failed'.
